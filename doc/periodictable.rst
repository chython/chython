chython\.periodictable package
==============================

Contains classes of all elements used in containers: common for `MoleculeContainer`, `Dynamic` for `CGRContainer`,
`Query` for `QueryContainer`.
Also available 3 special query atom types: `AnyElement`, `AnyMetal` and `ListElement`
